# Contexto do Projeto

Rods Themes e uma extensao de temas para VS Code.
Ela entrega 10 temas dark/neon inspirados em selecoes da Copa.

## Decisoes

- Nome publico: Rods Themes.
- Fonte dos temas: TypeScript com gerador.
- Visual: neon intenso sobre preto absoluto.
- Teste local: instalar VSIX pelo VS Code.
