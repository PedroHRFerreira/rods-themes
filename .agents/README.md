# .agents

Guarda instrucoes locais para agentes trabalharem neste projeto.
Esta pasta nao deve ser publicada nem enviada ao GitHub.

## Pastas

- `checklists`: passos de execucao e validacao.
- `context`: contexto do produto e decisoes atuais.
- `rules`: padroes de codigo e arquitetura.
- `skills`: fluxos especificos para geracao de temas.
